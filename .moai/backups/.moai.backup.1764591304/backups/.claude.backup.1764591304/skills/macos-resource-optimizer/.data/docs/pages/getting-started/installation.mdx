# Installation Guide

Complete step-by-step installation instructions for macOS Resource Optimizer v7.0.

## System Requirements

### Minimum Requirements
- macOS 10.15 (Catalina)
- Python 3.9
- 2GB RAM
- 100MB disk space

### Recommended Requirements
- macOS 13+ (Ventura or newer)
- Python 3.11+
- 4GB RAM
- SSD storage

## Installation Methods

### Method 1: Using UV (Recommended)

```bash
# Navigate to project directory
cd .claude/skills/macos-resource-optimizer/.data

# Create virtual environment
uv venv

# Activate environment
source .venv/bin/activate  # macOS/Linux
# OR
.venv\Scripts\activate     # Windows

# Install dependencies
uv pip install -r requirements.txt
```

### Method 2: Using pip

```bash
cd .claude/skills/macos-resource-optimizer/.data

# Create virtual environment
python3 -m venv .venv

# Activate environment
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

### Method 3: Using conda

```bash
cd .claude/skills/macos-resource-optimizer/.data

# Create conda environment
conda env create -f environment.yml

# Activate environment
conda activate macos-optimizer
```

## Troubleshooting Installation

### Issue: Python not found

```bash
# Verify Python installation
python3 --version

# If not installed, use Homebrew
brew install python@3.11
```

### Issue: Permission denied

```bash
# Grant execute permissions
chmod +x .claude/skills/macos-resource-optimizer/.data/bin/*
```

### Issue: Module not found

```bash
# Reinstall dependencies
pip install --upgrade -r requirements.txt
```

## Advanced Installation

### Development Installation

```bash
# Clone with development dependencies
git clone <repository>
cd .claude/skills/macos-resource-optimizer/.data

# Install with dev tools
uv pip install -r requirements-dev.txt

# Run tests
pytest tests/ -v
```

### Docker Installation

```bash
# Build Docker image
docker build -t macos-optimizer:7.0 .

# Run container
docker run -it macos-optimizer:7.0 /macos-resource-optimizer:1-analyze
```
