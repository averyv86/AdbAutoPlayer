# Command 2: optimize

Apply targeted optimizations to improve system performance.

## Syntax

```bash
/macos-resource-optimizer:2-optimize [options]
```

## Description

The optimize command:
- Plans optimization based on system analysis
- Applies targeted improvements
- Monitors impact in real-time
- Generates detailed change log
- Safe, reversible modifications

## Basic Usage

```bash
# Run optimization
/macos-resource-optimizer:2-optimize
```

Expected output:
```json
{
  "status": "success",
  "optimizations_applied": 12,
```

## Options

### Optimization Strategy

```bash
# Conservative strategy (minimal changes)
/macos-resource-optimizer:2-optimize --strategy conservative

# Balanced strategy (recommended)
/macos-resource-optimizer:2-optimize --strategy balanced

# Aggressive strategy (maximum optimization)
/macos-resource-optimizer:2-optimize --strategy aggressive
```

### Safety Options

```bash
# Dry run (preview changes)
/macos-resource-optimizer:2-optimize --dry-run

# Require confirmation
/macos-resource-optimizer:2-optimize --confirm

# Unsafe optimizations allowed
/macos-resource-optimizer:2-optimize --unsafe
```

### Focus Options

```bash
# Optimize specific component
/macos-resource-optimizer:2-optimize --component cpu

# Multiple components
/macos-resource-optimizer:2-optimize --component cpu --component memory

# All components
/macos-resource-optimizer:2-optimize --all
```

### Output Options

```bash
# JSON output
/macos-resource-optimizer:2-optimize --json

# Detailed output
/macos-resource-optimizer:2-optimize --verbose

# Quiet mode
/macos-resource-optimizer:2-optimize --quiet
```

## Optimization Strategies

### Conservative Strategy

Best for:
- Production systems
- Critical services
- Stability-first scenarios

Changes:
- Minimal system modifications
- Safe, proven optimizations
- Low risk

### Balanced Strategy (Default)

Best for:
- General use
- Most users
- Mixed workloads

Changes:
- Standard optimizations
- Moderate risk
- Good performance/safety balance

### Aggressive Strategy

Best for:
- Gaming/high-performance needs
- Development systems
- Maximum optimization

Changes:
- Extensive optimizations
- Higher risk
- Maximum performance

## Change Log

All changes are logged and reversible:

```bash
# View optimization history
/macos-resource-optimizer:2-optimize --show-history

# Revert last optimization
/macos-resource-optimizer:2-optimize --revert

# Revert all optimizations
/macos-resource-optimizer:2-optimize --revert-all
```

## Troubleshooting

### Optimization Failed

```bash
# View detailed error
/macos-resource-optimizer:2-optimize --verbose

# Revert to safe state
/macos-resource-optimizer:2-optimize --revert
```

### No Improvement Detected

```bash
# Check system load
/macos-resource-optimizer:1-analyze

# Try more aggressive strategy
/macos-resource-optimizer:2-optimize --strategy aggressive
```

### System Issues After Optimization

```bash
# Automatic rollback
/macos-resource-optimizer:2-optimize --revert

# Manual revert to previous state
/macos-resource-optimizer:2-optimize --revert-all
```
