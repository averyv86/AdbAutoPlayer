# Getting Started

Welcome to macOS Resource Optimizer v7.0! This guide will help you get up and running quickly.

## Prerequisites

- macOS 10.15+ (Catalina or later)
- Python 3.9 or higher
- Claude Code CLI
- MoAI-ADK 0.30.2 or higher

## Installation Steps

1. **Install Python Engine**
   ```bash
   cd .claude/skills/macos-resource-optimizer/.data
   uv venv
   source .venv/bin/activate
   uv pip install -r requirements.txt
   ```

2. **Validate Setup**
   ```bash
   /macos-resource-optimizer:0-init
   ```

3. **Run First Analysis**
   ```bash
   /macos-resource-optimizer:1-analyze
   ```

## Expected Output

A successful initialization returns:
```json
{
  "status": "success",
  "execution_time_seconds": 1.2,
  "system_info": {
```
