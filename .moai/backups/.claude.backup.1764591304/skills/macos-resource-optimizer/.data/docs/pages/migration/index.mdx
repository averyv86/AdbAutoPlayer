## Upgrade Path

### Step 1: Backup Configuration

```bash
# Backup current configuration
cp .claude/skills/macos-resource-optimizer/.data/config.json .claude/skills/macos-resource-optimizer/.data/config.v6.backup.json

# Backup any custom profiles
cp .claude/skills/macos-resource-optimizer/.data/profiles.json .claude/skills/macos-resource-optimizer/.data/profiles.v6.backup.json
```

### Step 2: Backup Data

```bash
# Backup metrics and logs
cp -r .claude/skills/macos-resource-optimizer/.data/logs .claude/skills/macos-resource-optimizer/.data/logs.v6.backup
cp -r .claude/skills/macos-resource-optimizer/.data/metrics .claude/skills/macos-resource-optimizer/.data/metrics.v6.backup
```

### Step 3: Uninstall v6

```bash
# Remove v6 installation
rm -rf .claude/skills/macos-resource-optimizer/.data/

# Clean v6 configuration
rm -rf ~/.config/macos-optimizer/
```

### Step 4: Install v7

```bash
# Create v7 directory
mkdir -p .claude/skills/macos-resource-optimizer/.data

# Navigate to project
cd .claude/skills/macos-resource-optimizer/.data

# Initialize with v7
/macos-resource-optimizer:0-init
```

### Step 5: Migrate Configuration

```bash
# Review old configuration
cat .claude/skills/macos-resource-optimizer/.data/config.v6.backup.json

# Apply settings to new config.json
nano .claude/skills/macos-resource-optimizer/.data/config.json

# Validate new configuration
/macos-resource-optimizer:0-init --validate-config
```

### Step 6: Verify Installation

```bash
# Test v7 installation
/macos-resource-optimizer:1-analyze

# Verify all experts working
/macos-resource-optimizer:0-init --diagnose
```


## Troubleshooting Migration

### Issue: "Old configuration not working"

**Solution**:

```bash
# Reset to v7 defaults
/macos-resource-optimizer:0-init --reset-config

# Manually migrate settings
nano .claude/skills/macos-resource-optimizer/.data/config.json

# Validate
/macos-resource-optimizer:0-init --validate-config
```

### Issue: "Command format unknown"

**Solution**: Use new v7 command format:

```bash
# OLD (v6): macos-optimizer init
# NEW (v7):
/macos-resource-optimizer:0-init
```

### Issue: "Metrics from v6 not visible"

**Solution**: v7 uses new metrics format:

```bash
# Start fresh with v7
/macos-resource-optimizer:1-analyze

# Export to compare
/macos-resource-optimizer:4-report --format json
```

### Issue: "Some features missing"

**Solution**: Check what's new:

- **Reporting**: `/macos-resource-optimizer:4-report`
- **Battery**: `/macos-resource-optimizer:2-optimize --component battery`
- **Network**: `/macos-resource-optimizer:2-optimize --component network`
- **Thermal**: `/macos-resource-optimizer:2-optimize --component thermal`

## Reverting to v6

If you need to downgrade:

```bash

# Restore v6 from backup
cp -r .claude/skills/macos-resource-optimizer/.data.v6.backup/ .claude/skills/macos-resource-optimizer/.data/

# Restore configuration
cp .claude/skills/macos-resource-optimizer/.data/config.v6.backup.json .claude/skills/macos-resource-optimizer/.data/config.json
```
