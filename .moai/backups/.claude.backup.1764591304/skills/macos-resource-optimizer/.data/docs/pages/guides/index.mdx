## Quick Problem Solver

### Slow Performance

1. Run analysis: `/macos-resource-optimizer:1-analyze`
2. Check recommendations
3. Apply optimizations: `/macos-resource-optimizer:2-optimize`
4. Monitor results: `/macos-resource-optimizer:3-monitor`

### High Memory Usage

```bash
# Analyze memory
/macos-resource-optimizer:1-analyze --category memory

# Optimize memory
/macos-resource-optimizer:2-optimize --component memory
```

### Thermal Issues

```bash
# Check thermal status
/macos-resource-optimizer:1-analyze --category thermal

# Apply thermal optimizations
/macos-resource-optimizer:2-optimize --component thermal
```

### Battery Drain

```bash
# Analyze battery
/macos-resource-optimizer:1-analyze --category battery

# Optimize battery
/macos-resource-optimizer:2-optimize --component battery
```

## Optimization Workflows

### Daily Optimization

```bash
# Daily check and optimize
/macos-resource-optimizer:1-analyze && \
/macos-resource-optimizer:2-optimize --strategy conservative
```

### Weekly Performance Review

```bash
# Comprehensive analysis
/macos-resource-optimizer:1-analyze --detailed && \
/macos-resource-optimizer:4-report --period week
```

### Monthly Deep Dive

```bash
# Full analysis and optimization
/macos-resource-optimizer:1-analyze --very-detailed && \
/macos-resource-optimizer:2-optimize --strategy balanced && \
/macos-resource-optimizer:4-report --period month --format html
```

## Pro Tips

### Tip 1: Baseline First

Always establish baseline before optimizing:
```bash
/macos-resource-optimizer:1-analyze > baseline.json
```

### Tip 2: Conservative Start

Start with conservative strategy:
```bash
/macos-resource-optimizer:2-optimize --strategy conservative
```

### Tip 3: Monitor Changes

Monitor after optimization:
```bash
/macos-resource-optimizer:3-monitor --duration 3600
```

### Tip 4: Schedule Runs

Use cron to schedule regular runs:
```bash
# Daily at 2 AM
0 2 * * * /macos-resource-optimizer:1-analyze
```

### Tip 5: Export Data

Export metrics for analysis:
```bash
/macos-resource-optimizer:4-report --format csv --output metrics.csv
```

## Integration Examples

### With cron

Schedule regular optimizations:

```bash
# Edit crontab
crontab -e

# Add daily optimization (2 AM)
0 2 * * * /macos-resource-optimizer:1-analyze && /macos-resource-optimizer:2-optimize
```

### With shell scripts

Create optimization scripts:

```bash
#!/bin/bash
echo "Starting daily optimization..."
/macos-resource-optimizer:0-init
/macos-resource-optimizer:1-analyze
/macos-resource-optimizer:2-optimize
/macos-resource-optimizer:4-report
echo "Done!"
```

### With Docker

Run in container:

```bash
docker run -it macos-optimizer:7.0 /macos-resource-optimizer:1-analyze
```
